# 🧠 소스 코드

이 폴더에는 프로젝트의 **메인 소스 코드**가 포함되어 있습니다.

## 📌 주요 내용

| 파일/폴더 이름        | 설명                            |
|----------------------|---------------------------------|
| `main.c` 등           | 프로젝트의 핵심 동작을 구성하는 코드들 |
| `inc/`, `src/`, `lib/` | 헤더, 소스, 외부 라이브러리 디렉터리 (존재하는 경우) |

> 💡 이 코드는 STM32 기반 ARM 프로젝트용으로 작성되었으며, `STM32CubeIDE` 또는 `Keil` 환경에서 빌드 가능합니다.
